# Técnicas e Instrumentación en Psicofisiologia | Licenciatura en Psicología | Universidad Autónoma de Baja California

## Introducción

En este repositorio se encuentran los cuadernos Jupyter que se utilizarán para prácticas asociadas con el material de estudio.

Se estructuran en cinco tópicos:

- Introducción a Python, ambientes de desarrollo y manejo de datos
- Obtención y preprocesamiento de datos fisiológicos y conductuales con Python
- Selección de tareas experimentales en neuropsicología e implementación con PsychoPy
- Selección e implementación de _pipelines_ de análisis estadístico
- Selección e implementación de _pipelines_ de análisis en bioseñales
